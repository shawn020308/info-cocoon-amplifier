# Changelog

## 0.3.0 — 重写

UI 层全面重构。删掉了分散的模块，代之以一个内聚的面板。设计风格从功能堆砌转向克制。

### 视觉

- **三套主题**：Claude 温润 / GitHub 锐利 / Dark Modern。通过设置面板即时切换。
- **全局字体缩放**，0.8 到 1.5 倍连续调节。
- 面板 Tab 精简为三个：设置、统计（含黑名单管理）、学习（含知识库编辑）。
- 折叠模式新增 **dim**：不遮挡、不标记，仅将违规评论弱化为一行省略号。悬停可见。

### 代码

- `fold-ui.ts`、`manual-blacklist.ts`、`logger.ts` 删除，逻辑并入 `ui.ts`。
- `ui.ts` 现在是唯一的 UI 入口，包含面板、FAB、Toast、评论折叠和拉黑按钮的全部实现。
- 内部日志统一为英文。
- `alert` / `confirm` 替换为自绘 Toast。

### 文档

- README 重写。去掉多余的 badge 和感叹号，只说清楚两件事：这是什么、怎么用。

### 兼容

- 旧配置文件中缺失 `theme` 或 `fontScale` 字段时，自动回退为 `github` 主题和 `1.0` 倍缩放。
- 旧 `filterDimensions` 字段继续兼容解析。

## 0.2.4 — 学习

- AI 自我学习：取消拉黑、误判展开、手动拉黑行为自动累积，攒满 20 条后 AI 生成过滤画像。
- 知识库：手动添加语境知识，注入 System Prompt 辅助判定反讽和特定称呼。
- 用户画像：AI 生成 300 字过滤画像，可手动编辑或重新生成。

## 0.2.3 及之前

- 初始版本，包含 DOM 扫描、AI 判定、黑名单、缓存、四种折叠模式等基础能力。
